# GO TOUR - Project Context

## Overview
GO TOUR is a premium, mobile-first travel companion application. 
Phase 1 focuses on the onboarding experience: Welcome, Login, and Registration.

## Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Vanilla CSS (CSS Modules + Global Variables)
- **Icons:** Lucide React
- **Router:** React Router DOM

## Development Principles
- **Mobile-First:** Design for small screens first, then scale up.
- **Mock Data:** No real backends in Phase 1. Use timeouts to simulate network requests.
- **Premium UI:** High-quality animations, glassmorphism, and smooth transitions.
