# UI Rules & Design System

## Colors
- **Primary:** `#3b82f6` (Modern Blue)
- **Gradient:** `linear-gradient(135deg, #2563eb, #7c3aed)`
- **Secondary/Success:** `#10b981`
- **Error:** `#ef4444`
- **Background:** `#f8fafc`
- **Text Main:** `#1e293b`
- **Text Muted:** `#64748b`

## Typography
- **Font Family:** Inter, system-ui, sans-serif
- **Weights:** 400 (Regular), 500 (Medium), 700 (Bold)

## Dimensions
- **Border Radius:**
  - sm: 6px
  - md: 12px
  - lg: 20px
- **Spacing:** multiples of 4px (xs=4, sm=8, md=16, lg=24, xl=32)

## Animations
- **Fade In:** 0.5s ease-out
- **Hover:** Transform translateY(-1px) + Shadow elevation
